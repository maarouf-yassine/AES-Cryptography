These 16 bytes of the plaintext are then arranged into a 4 x 4 matrix. From this matrix we will do the 10 rounds of the AES algorithm if the key has 128 bits (if the key has 192 bits, it's 12 rounds, and when the key has 256 bits, it's 14 rounds). Each round has 4 steps, Byte Substitution, Row Shifting, Column Mixing, then adding the Key for that round.
